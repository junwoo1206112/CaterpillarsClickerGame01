# 🎮 게임씬 Phase 1-2 통합 설정 가이드

## 📋 방법 1: 자동 설정 (권장)

### 1. 메뉴에서 실행
```
Unity 메뉴 > Tools > Game > Setup Game Scene
```

### 2. 확인 대화상자
```
"This will create all necessary game objects for Phase 1-2."
→ "Yes" 클릭
```

### 3. 자동 생성되는 것들
```
✅ DataManager (Phase 1)
✅ GameplayManager (Phase 2)
✅ Canvas + UI 요소
   ├── ScoreText
   ├── TouchCountText
   ├── StageText
   ├── MultiplierText
   ├── MessageText
   ├── ClickButton
   ├── SpeedBoostButton
   └── ResetButton
✅ Player (캐릭터 - 구체)
✅ EventSystem
✅ GameUI 컴포넌트
```

## 📋 방법 2: 수동 설정

### 1. DataManager 생성
```
1. Hierarchy 우클릭 > Create Empty
2. 이름: "DataManager"
3. Add Component > DataManager
```

### 2. GameplayManager 생성
```
1. Hierarchy 우클릭 > Create Empty
2. 이름: "GameplayManager"
3. Add Component > GameplayManager
```

### 3. Canvas 생성
```
1. Hierarchy 우클릭 > UI > Canvas
2. Canvas 설정:
   - Render Mode: Screen Space - Overlay
   - Canvas Scaler: Scale With Screen Size
   - Reference Resolution: 1920x1080
```

### 4. UI 텍스트 생성 (4 개)
```
1. Canvas 우클릭 > UI > Text - TextMeshPro (또는 Legacy)
2. 이름: "ScoreText"
3. 위치: (20, -20), 크기: (300, 30)
4. Text: "Score: 0", Font Size: 24

반복:
- TouchCountText: (20, -50), "Touches: 0"
- StageText: (20, -80), "Stage: 1"
- MultiplierText: (20, -110), "Multiplier: 1x"
- MessageText: (20, -140), "" (빈 문자열)
```

### 5. 버튼 생성 (3 개)
```
1. Canvas 우클릭 > UI > Button
2. 이름: "ClickButton"
3. 위치: (100, -200), 크기: (150, 50)
4. Button 텍스트: "Click!"

반복:
- SpeedBoostButton: (260, -200), "Speed Boost"
- ResetButton: (420, -200), "Reset"
```

### 6. Player 생성
```
1. Hierarchy 우클릭 > 3D Object > Sphere
2. 이름: "Player"
3. 위치: (0, 1, 5)
4. 크기: (2, 2, 2)
5. Add Component:
   - ClickHandler
   - CharacterEvolution
6. BoxCollider 확인 (IsTrigger = true)
7. MeshRenderer 색상: 초록색
```

### 7. GameUI 컴포넌트 추가
```
1. Canvas 선택
2. Add Component > GameUI
3. UI References 연결:
   - Score Text: ScoreText
   - Touch Count Text: TouchCountText
   - Stage Text: StageText
   - Multiplier Text: MultiplierText
   - Message Text: MessageText
```

### 8. 이벤트 연결
```
ClickButton:
1. ClickButton 선택
2. Button 컴포넌트 > On Click()
3. + 버튼 클릭
4. Player 드래그
5. 함수: ClickHandler.HandleClick()

SpeedBoostButton:
1. SpeedBoostButton 선택
2. Button 컴포넌트 > On Click()
3. + 버튼 클릭
4. GameplayManager 드래그
5. 함수: GameplayManager.ActivateSpeedBoost()

ResetButton:
1. ResetButton 선택
2. Button 컴포넌트 > On Click()
3. + 버튼 클릭
4. GameplayManager 드래그
5. 함수: GameplayManager.ResetGame()
```

## 📋 데이터 연동

### 1. ScriptableObject 확인
```
Assets/ScriptableObjects/
├── Character/
│   └── CharacterList.asset
├── TouchFunction/
│   └── TouchFunctionList.asset
├── Item/
│   └── ItemList.asset
└── Config/
    └── ConfigList.asset
```

### 2. Excel 데이터 변환
```
Tools > Data > Create Excel Template
Tools > Data > Convert Excel to ScriptableObject
```

### 3. DataManager 자동 로드
```
DataManager.cs 에 Start() 메서드 추가됨:
private void Start()
{
    Invoke(nameof(LoadData), 0.5f);
}
```

## 🧪 테스트

### Play 모드 시작
```
1. 저장 (Ctrl+S)
2. Play 버튼 클릭
3. Console 에서 로그 확인
```

### Console 로그 예시
```
[DataManager] DI Container initialized
[DataManager] Loaded 3 characters
[DataManager] Loaded 3 touch functions
[TouchFunctionManager] Initialized 3 functions from data
[GameSetup] GameplayManager setup complete
```

### 클릭 테스트
```
1. Player(구체) 클릭
2. 또는 "Click!" 버튼 클릭
3. Score 증가 확인
4. UI 업데이트 확인
```

### 진화 테스트
```
1. 100 점까지 클릭 → Stage 2 (번데기)
2. 500 점까지 클릭 → Stage 3 (나비)
3. Console 에서 진화 로그 확인
```

### 터치 기능 테스트
```
1. 50 회 클릭 → "Bonus Enabled!"
2. 3 회마다 → "Bonus Click Activated!"
3. "Speed Boost" 버튼 클릭 → 2 배속 60 초
```

## 🐛 문제 해결

### Q1. UI 가 안 보임
```
해결: Canvas 의 Render Mode 가 Screen Space - Overlay 인지 확인
```

### Q2. 클릭이 안 됨
```
해결: Player 의 BoxCollider 가 있는지 확인 (IsTrigger = true)
```

### Q3. 점수가 안 오름
```
해결: GameplayManager 와 ScoreManager 연결 확인
```

### Q4. 진화가 안 됨
```
해결: CharacterListSO 가 할당되었는지 확인
```

### Q5. Console 에 에러
```
해결: 
1. 에러 메시지 확인
2. 줄 번호에서 문제 찾기
3. null 참조 확인
```
