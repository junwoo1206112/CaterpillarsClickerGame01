# Phase3: UI 시스템 구현계획

## 1. 개요
- **목표**: 게임 UI 및 설정 시스템 구현
- **기간**: 3 주차
- **핵심 기능**: 상단 UI 5 개, 설정, 커스터마이징

## 2. 주요 기능
### 2.1 상단 UI (좌측 상단)
1. **설정 버튼**: 사운드/볼륨 조절
2. **캐릭터 색상 변경**: 스킨 시스템
3. **배경 변경**: 배경 스킨 시스템
4. **스피드 부스터**: 1 분 2 배속
5. **아이템**: 클릭 50 증가 사용

### 2.2 설정 시스템
- 사운드 온/오프
- BGM 볼륨
- SFX 볼륨
- 저장/불러오기

### 2.3 커스터마이징
- 캐릭터 색상 프리셋
- 배경 이미지 교체
- 잠금 해제 시스템

## 3. 기술 스펙
- Unity UI (UGUI)
- MVVM 패턴 (UI 상태 관리)
- PlayerPrefs/JSON 저장

## 4. 테스트 계획
- UI 반응형 테스트
- 설정 저장/불러오기 테스트
- 볼륨 조절 테스트
