# 세부 스펙: 엑셀 데이터 시스템 (DI 기반)

## 1. 목적
- 게임 데이터를 엑셀로 관리하고 ScriptableObject 로 자동 변환
- 확장 가능한 DI 아키텍처로 결합도 낮춤
- 에디터에서 원클릭 데이터 생성/변환

## 2. 인터페이스 설계

### 2.1 IDataService<T>
```csharp
public interface IDataService<T>
{
    Task<T> LoadAsync(string path);
    Task SaveAsync(string path, T data);
    bool Validate(T data);
}
```

### 2.2 IExcelConverter
```csharp
public interface IExcelConverter
{
    void CreateTemplateExcel(string outputPath);
    ScriptableObject ConvertToScriptableObject(string excelPath);
}
```

## 3. 구현 클래스

### 3.1 ExcelDataService<T>
- NPOI 사용 (.xlsx 읽기/쓰기)
- 제네릭으로 다양한 데이터 타입 지원
- 에러 핸들링 포함

### 3.2 DataContainer
```csharp
public class DataContainer
{
    private readonly Dictionary<Type, object> _services = new();
    
    public void Register<T>(T service) where T : class;
    public T Resolve<T>() where T : class;
}
```

## 4. 엑셀 시트 구성

### 4.1 CharacterData 시트
| Column | Type | Description |
|--------|------|-------------|
| ID | string | 고유 ID (ex: CHAR_001) |
| Name | string | 캐릭터 이름 |
| Stage | int | 성장 단계 (1:애벌레, 2:번데기, 3:나비) |
| ScoreRequired | int | 필요 점수 |
| SpritePath | string | 스프라이트 경로 |
| Color | string | HEX 색상값 |

### 4.2 BackgroundData 시트
| Column | Type | Description |
|--------|------|-------------|
| ID | string | 배경 ID |
| Name | string | 배경 이름 |
| SpritePath | string | 스프라이트 경로 |
| UnlockScore | int | 해금 점수 |

### 4.3 ItemData 시트
| Column | Type | Description |
|--------|------|-------------|
| ID | string | 아이템 ID |
| Name | string | 아이템 이름 |
| Effect | string | 효과 타입 |
| Value | int | 효과 값 |
| Duration | float | 지속시간 (초) |
| IconPath | string | 아이콘 경로 |

### 4.4 TouchFunctionData 시트
| Column | Type | Description |
|--------|------|-------------|
| ID | string | 기능 ID |
| TriggerCount | int | 발동 조건 (터치 횟수) |
| EffectType | string | 효과 타입 |
| Multiplier | float | 배율 |
| Duration | float | 지속시간 |
| Cooldown | float | 쿨다운 |

### 4.5 ConfigData 시트
| Column | Type | Description |
|--------|------|-------------|
| Key | string | 설정 키 |
| Value | string | 설정 값 |
| Description | string | 설명 |

## 5. MenuItem 기능

### 5.1 빈 엑셀 생성
```
Tools > Data > Create Excel Template
- Assets/ExcelData/GameData.xlsx 생성
- 위 5 개 시트와 컬럼 자동 생성
```

### 5.2 데이터 컨버팅
```
Tools > Data > Convert Excel to ScriptableObject
- 엑셀 읽어서 각 ScriptableObject 생성
- Assets/ScriptableObjects/ 하위에 저장
```

## 6. 검증 규칙
- ID 중복 체크
- 필수 컬럼 공백 체크
- 타입 불일치 체크
- 경로 존재 여부 체크

## 7. 에러 처리
- 유효성 실패 시 Console 에 Error 로그
- 변환 중단 및 롤백
- 상세 에러 리포트 생성
