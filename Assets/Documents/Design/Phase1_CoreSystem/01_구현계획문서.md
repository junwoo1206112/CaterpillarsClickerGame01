# Phase1: 핵심 시스템 구현계획

## 1. 개요
- **목표**: 클리커 게임의 기본 골격 및 DI 기반 데이터 시스템 구축
- **기간**: 1 주차
- **핵심 기능**: 캐릭터 클릭 → 점수 획득 → 성장 시스템

## 2. 주요 기능
1. 캐릭터 클릭 시스템
2. 점수 및 레벨 관리
3. 캐릭터 성장 (애벌레 → 번데기 → 나비)
4. DI 기반 데이터 관리 시스템
5. 엑셀 데이터 컨버팅 시스템

## 3. 기술 스펙
- Unity 2022+
- C# 9.0+
- NPOI (엑셀 처리)
- Dependency Injection 패턴
- ScriptableObject 기반 데이터 저장

## 4. 폴더 구조
```
Assets/
├── Documents/
│   └── Design/
│       ├── Phase1_CoreSystem/
│       ├── Phase2_Gameplay/
│       ├── Phase3_UI/
│       └── Phase4_Content/
├── Scripts/
│   ├── Core/ (DI 컨테이너, 인터페이스)
│   ├── Data/ (데이터 서비스, 모델)
│   ├── Managers/ (게임 매니저)
│   ├── Gameplay/ (클릭, 성장 로직)
│   ├── UI/ (UI 컨트롤러)
│   └── Utilities/ (헬퍼, 확장메서드)
├── ScriptableObjects/
│   ├── Character/
│   ├── Stage/
│   ├── Item/
│   └── Config/
├── ExcelData/
└── Editor/
```

## 5. 테스트 계획
- 단위 테스트: 데이터 로드/저장
- 통합 테스트: 클릭 → 점수 → 성장 흐름
- 자동화 테스트: 엑셀 컨버팅
