# 세부 스펙: 터치 기능 시스템

## 1. 목적
- 확장 가능한 터치 기능 아키텍처
- 터치 카운트 기반 자동 발동
- 전략 패턴으로 기능 추가 용이

## 2. 인터페이스

### 2.1 ITouchFunction
```csharp
public interface ITouchFunction
{
    int TriggerCount { get; }
    float Multiplier { get; }
    float Duration { get; }
    float Cooldown { get; }
    
    bool CanActivate(int currentCount, float lastActivateTime);
    TouchEffect Activate();
}
```

### 2.2 TouchEffect
```csharp
public struct TouchEffect
{
    public int BonusClicks;
    public float ScoreMultiplier;
    public float Duration;
    public string EffectName;
}
```

## 3. 구현 기능

### 3.1 BasicTouchFunction
- 기본 터치 처리
- 1 회 터치 = 1 점

### 3.2 BonusTouchFunction (50 회 이상)
- 조건: 총 터치 50 회 이상
- 효과: 3 회마다 1 회 추가 (실제 3 클릭 → 4 점)
- 자동 발동

### 3.3 SpeedBoostFunction
- 조건: 아이템/버튼 사용
- 효과: 1 분간 2 배
- 쿨다운: 5 분

### 3.4 CriticalTouchFunction
- 조건: 확률 기반
- 효과: 5 배 점수
- 확률: 10%

## 4. TouchService

```csharp
public class TouchService : ITouchService
{
    private List<ITouchFunction> _functions = new();
    private int _totalTouchCount;
    
    public void RegisterFunction(ITouchFunction function);
    public TouchResult ProcessTouch(Vector2 position);
    public int GetTotalTouchCount();
}
```

## 5. 데이터 연동
- TouchFunctionData 시트에서 로드
- ScriptableObject 로 캐시
- 런타임에서 동적 등록

## 6. 검증 항목
- 터치 카운트 정확도
- 동시 발동 처리
- 쿨다운 관리
- 효과 중첩
