# Phase2: Gameplay 기능 구현계획

## 1. 개요
- **목표**: 터치 기반 기능 및 아이템 시스템 구현
- **기간**: 2 주차
- **핵심 기능**: 터치 카운트 기반 버프, 아이템 사용

## 2. 주요 기능
### 2.1 터치 기능 시스템
- 터치 카운트 추적
- 50 회 이상 터치 시: 3 회마다 1 회 추가 클릭 효과
- 기능 확장 가능한 아키텍처 (전략 패턴)

### 2.2 아이템 시스템
- 터치 시 클릭 50 증가 아이템
- 아이템 사용 UI
- 아이템 소모 로직

### 2.3 스피드 부스터
- 1 분간 2 배속 터치
- 쿨다운 관리
- UI 표시

## 3. 기술 스펙
- 전략 패턴: 터치 기능 확장
- Observer 패턴: 이벤트 기반 통신
- ScriptableObject: 아이템 데이터 정의

## 4. 테스트 계획
- 터치 카운트 정확도 테스트
- 버프 중첩 테스트
- 아이템 사용/소모 테스트
